Semantic Search for Courses

This project uses Open AI Ada embeddings to find courses with similar descriptions. Inspired by the Cardinal Compass project at Stanford.

Development steps

1. Get list of courses and their descriptions at IU using web scraping (Python, Selenium, BeautifulSoup)

2. Use OpenAI API to get embeddings for each course description (Python, OpenAI API)

3. Store embeddings in a database

4. Create a web interface to search for similar courses (Python-Flask, Azure)

Future steps

1. Add more courses from other departments

2. Add more features to the search interface including filtering by department, course number, etc.

3. Add more information about each course including prerequisites, credit hours, etc.
